# 📤 如何上传 script.js 到 GitHub

## 🎯 目标
上传修改后的 `script.js` 文件到你的 GitHub 仓库

---

## 📋 方式 A：GitHub Web 界面（推荐）

### 第 1 步：打开你的 GitHub 仓库
访问：https://github.com/snala4451/my-website

### 第 2 步：点击 "Add file"
在仓库页面右上方，点击绿色的 "Add file" 按钮

### 第 3 步：选择 "Upload files"
从下拉菜单中选择 "Upload files"

### 第 4 步：上传 script.js
- 拖拽 `script.js` 文件到上传区域
- 或点击 "choose your files" 选择文件

### 第 5 步：提交更改
1. 在下方输入提交信息：`Update script.js - fix mobile parsing`
2. 点击 "Commit changes" 按钮

### 第 6 步：等待更新
- GitHub Pages 通常在 1-2 分钟内更新
- 你可以在 "Deployments" 中查看部署状态

### 第 7 步：测试
1. 在手机上打开你的网站
2. 输入任何抖音或小红书链接
3. 点击"能量启动"
4. 应该立即看到解析结果

---

## 📋 方式 B：使用上传工具

### 第 1 步：打开上传工具
打开 `GITHUB_UPLOAD.html` 文件

### 第 2 步：填写信息
- GitHub 用户名：`snala4451`
- 仓库名称：`my-website`

### 第 3 步：生成步骤
点击 "生成上传步骤" 按钮

### 第 4 步：按照步骤上传
按照生成的步骤在 GitHub 网站上完成上传

---
